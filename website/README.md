# USYD 高分辅导预约网页

这是一个为 USYD Data Science 与 FMS 专业学生设计的个人辅导介绍及预约网页。网站以紫色为主视觉，集中展示老师的学术背景、辅导特色、学生成果和可预约时间，帮助学生快速了解课程并完成约课。

![网页主视觉](study-portrait.png)

## 主要功能

- **个人介绍**：展示 USYD Bachelor of Science 学历、专业成绩、Dalyell Scholar Program、Dean's List、奖学金及 Kaggle 竞赛经历。
- **辅导特色**：强调考点归纳、作业与项目辅导、考前冲刺，以及清晰、有鼓励感的教学方式。
- **学生成果**：展示 HD、Quiz 满分、Assignment 满分及小组作业高分案例。
- **空闲时间管理**：老师可以自主开放或关闭可预约时段。
- **在线预约**：学生可选择老师开放的日期和时间，并填写课程与辅导需求。
- **打印或保存 PDF**：预约信息生成后，可通过浏览器打印或另存为 PDF。

## 技术构成

本项目为纯静态网页，无需安装框架或依赖：

- HTML5
- CSS3
- 原生 JavaScript
- 浏览器 `localStorage`，用于保存演示版的空闲时间设置

## 本地查看

下载或克隆仓库后，直接用浏览器打开 `website/index.html` 即可预览。

## 使用说明

1. 老师点击“管理空闲时间”，选择并保存可预约时段。
2. 学生在预约区域选择日期和时间，填写课程及辅导需求。
3. 提交后生成预约摘要，可复制、打印或保存为 PDF。

## 当前版本说明

当前为前端演示版：

- 预约数据不会自动发送给老师。
- 空闲时间仅保存在当前浏览器设备中。
- 清除浏览器数据或更换设备后，本地设置可能丢失。
- 后续可接入数据库、登录系统、邮件通知或日历同步，实现多人在线使用。

## 隐私说明

网页不会主动上传学生填写的信息。提交预约后，请通过与老师约定的联系方式发送预约摘要。
